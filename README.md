# Commute

Compare driving, transit, and drive+transit (GO) routes between two locations using Google Maps.

## Setup

1. Copy `.env.example` to `.env`.
2. Add your Google Maps API key to `.env` as `VITE_GOOGLE_MAPS_API_KEY`.
3. In Google Cloud Console, enable: Maps JavaScript API, Places API, and Directions API (Legacy).
4. Run `npm install` then `npm run dev`.
